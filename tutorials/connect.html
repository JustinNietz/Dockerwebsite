How to Connect Docker with Pycharm

by Nikolett Muller

Let us see the steps we need to take to connect Docker with a Pycharm project.

1. Requirements

To start working, we need to check a couple of things beforehand.

Needed requirements:

  • Docker is installed
  • Pycharm Pro is installed
  • Git is installed

When all requirements are met, we need to start Docker Desktop and Pycharm Pro.

2. Creating a New Project

We create a new Python Project in Pycharm Pro. Then we go to Github and clone an existing repository.

Then we go back to Pycharm, click on VCS on top bar, then select Get from Version Controlâ€¦ After that, we paste the URL of the existing repository and we give a directory; finally, a quick click on to Clone button.
